# snake-term

A simple "snake" game in the terminal

![screenshot](https://raw.github.com/seven1m/snake-term/master/snake.png)

To play:

    $ ruby snake.rb

Use the Vim directional keys for movement:

    h = left
    j = down
    k = up
    l = right

Don't hit a wall or yourself!
